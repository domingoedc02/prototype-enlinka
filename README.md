# Enlinka

A prototype company website built with Next.js, Tailwind CSS, and Framer Motion.

## Tech Stack

- [Next.js](https://nextjs.org) 14
- [React](https://react.dev) 18
- [Tailwind CSS](https://tailwindcss.com) 4
- [Framer Motion](https://www.framer.com/motion/)
- [TypeScript](https://www.typescriptlang.org)
- [Lucide React](https://lucide.dev)

## Pages

- Home
- About
- Blog
- Careers
- Consultation
- Services (with dynamic slugs)

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org) (v18 or later)

### Installation

```bash
git clone git@github.com:domingoedc02/prototype-enlinka.git
cd prototype-enlinka
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

### Build & Production

```bash
npm run build
npm start
```

### Lint

```bash
npm run lint
```
